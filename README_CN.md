# 🚀 UNIRIG 一键安装器

在 ComfyUI 中一键安装、配置并启用 **UNIRIG**

---

## ⚡ 一键体验

无需手动配置，无需调试。

👉 只需：

**安装 UNIRIG → ONE-CLICK INSTALL → 打开 COMFYUI → 运行工作流**

---

## 🧠 功能说明

该工具会自动：

- 检测 ComfyUI 环境
- 安装 UniRig
- 创建独立运行环境（comfy-env）
- 安装所有必要组件
- 应用所有必要修复

👉 **结果：UniRig 可立即在 ComfyUI 中使用**

---

## 🚀 使用方法 -（推荐使用 Microsoft Edge、Firefox 或 Opera）

### 1. 安装 UniRig
点击 **"Install UniRig"**

### 2. 点击 ONE-CLICK INSTALL
执行自动安装

### 3. 打开 ComfyUI
启动 ComfyUI

### 4. 运行工作流
加载 UniRig 工作流并运行

👉 完成
