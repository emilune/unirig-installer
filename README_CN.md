# UniRig Installer

一键在 ComfyUI 中安装 UniRig —— 快速为你的 3D 角色绑定骨骼。

---

## ⚠️ 重要提示

安装 UniRig 后，**不要重启 ComfyUI**。

请务必**完全关闭 ComfyUI**，然后再运行 UniRig Installer。

❌ 不要重启 ComfyUI
✅ 必须完全关闭 ComfyUI

否则可能会创建错误的环境。

---

## ⚠️ 如果你之前已经安装过 UniRig

可能已经存在旧的环境，并导致问题。

最简单、最安全的方法是：

1. 打开 ComfyUI
2. 点击界面顶部的 **ENV** 按钮，打开 **ComfyUI-Env-Manager**
3. 删除已有的 UniRig 环境
4. 完全关闭 ComfyUI

然后再运行 UniRig Installer。

---

## 🚀 安装步骤

### 第一步 — 在 ComfyUI 中安装 UniRig

* 打开 ComfyUI
* 通过 Manager 安装 **UniRig**

👉 然后**完全关闭 ComfyUI**

---

### 第二步 — 运行 UniRig Installer

按顺序点击：

1. Detect
2. Update comfy-env
3. Generate Script
4. Export Script

---

### 第三步 — 运行脚本

在 PowerShell 中执行：

```powershell
.\SCRIPT_UNIRIG.ps1
```

---

### 第四步 — 启动 ComfyUI

* 启动 ComfyUI
* 加载 UniRig workflow
* 运行

---

## 💡 说明

* 本工具会自动配置环境
* 在部分最新 GPU 上可能会有差异
