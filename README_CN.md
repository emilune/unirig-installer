# UniRig Installer

一键在 ComfyUI 中安装 UniRig。

---

## 🌍 语言

[English](README.md) | [Français](README_FR.md) | 中文

---

## ⚠️ 重要

安装后 **不要重启 ComfyUI**  
必须 **完全关闭 ComfyUI**

---

## 🚀 安装步骤

### 第一步 — 安装 UniRig

- 打开 ComfyUI  
- 安装 UniRig  

👉 然后关闭 ComfyUI

---

## ⚠️ 如果之前安装过 UniRig

（如果是第一次安装可跳过）

步骤：

1. 打开 ComfyUI  
2. 点击 ENV  
3. 删除 UniRig 环境  
4. 关闭 ComfyUI  

---

### 第二步 — 运行 Installer

1. Detect  
2. Update comfy-env  
3. Install UniRig env  
4. Generate Script  
5. Export Script  

---

### 第三步 — 运行脚本

```powershell
.\SCRIPT_UNIRIG.ps1
```

---

### 第四步 — 启动 ComfyUI

运行 workflow

---

## 💡 提示

自动配置环境  
部分 GPU 可能不同
